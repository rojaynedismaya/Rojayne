HTML and C# and C++ and Phython Differences: Previously, the norm was to use HTML tags for styling web contents. For example: Some text. The ‘i’-tag here tells the browser to italicize ‘Some text’. However, CSS (Cascading Style Sheets) is now used to add styles to HTML. Pretty much all the styles you see on websites are directly or indirectly done with CSS. A CSS statement like: p{color:red;} - gives a red font color to all p-tags in HTML While HTML, CSS are rendered by browsers, Python, C++ and C# are actual programming languages. A programming language is a set of syntax and commands that are readable to the human being, and can be compiled or interpreted to the machine language for the instructions to be executed. Python is an interpreted language. It has an interpreter which ‘translates’ each command line-by-line on execution to machine language, and then execute the command. Python is regarded by many as the ideal first programming language. It has simpler syntax (relative to C++ and C#), supports all programming paradigms and is general purpose. In the context of web development, Python is more commonly used at the backend compared to C++ and C# with the famous frameworks django and flask. C++ is also a general purpose language. It is compiled rather than interpreted. A compiled language requires that all of the written instructions are compiled into machine code. Then the compiled code can then be executed over and over again. This is different from interpreted languages in the sense that for interpreted languages, the code(instructions) is interpreted each time it is executed. C++ has flourished most game engine and high-end game development, and also in cross platform Graphical User Interface applications. C# is regarded as the language for microsoft developers. Do not be misled though. It is also a compiled language. It can pretty much be used for whatever C++ is used for. C# is also used together with ASP.NET for backend web development. These days, it is trying to make a presence in mobile development. Generally, all programming languages can be used to do the same things. Just that there are degrees of advantage for certain languages in certain niches
